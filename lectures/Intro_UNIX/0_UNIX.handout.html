Directories (folders)

  • Listing files in a directory

    $ ls
$ ls -l

  • making a directory

    $ mkdir mynewdir

  • changing directories

    $ cd mynewdir # go in
$ cd ..       # go back
$ cd ~        # go HOME
$ cd          # go HOME
$ cd ../mynewdir/text # go to another directory

  • removing directories

    $ rmdir mynewdir

DESCRIPTION man is the system's manual pager. Each page argument given to man is normally the name of a program, utility or function. Command line programs

  • anatomy of running a program

  • the program name comes first. It can be just the name ls or the full path to the program /bin/ls

    $ ls
$ /bin/ls
  • arguments to the program come next. Some can be named with an argument like program -i input -o output
  • or the program can just have a set of arguments like cat file1 file2 file3

  • Instead of having output be printed to the screen it can be captured to a file

    $ ls > listing
$ more listing
$ cat file1 file2 file3 > all_files
